# Convolution Accelerator — HW-SW Codesign Midterm

FPGA accelerator for a CNN-based dog/cat image classifier, targeting the Pynq xc7z020 board.

## Repository Structure

```
├── HLS/
│   ├── HLS_Conv/          # Conv2D HLS source (conv2d.cpp, conv2d.h)
│   ├── HLS_MaxPool/       # MaxPool HLS source (maxpool.cpp, maxpool.h)
│   └── Scripts/
│       ├── Conv2D_HW_HLS_impl.tcl    # HLS synthesis + IP export for Conv2D
│       └── MaxPool_HW_HLS_impl.tcl   # HLS synthesis + IP export for MaxPool
├── Vivado/
│   └── Scripts/
│       ├── Conv2D_HW_Vivado.tcl      # Block design creation (PS + Conv2D + MaxPool IPs)
│       └── Conv2D_HW_Vivado_impl.tcl # Synthesis + implementation + bitstream
├── Solution/
│   └── src/               # ARM software (model.cpp, CConv2DProxy, CMaxPoolProxy, cnnSolver)
├── IP-repo/               # Exported HLS IPs (generated by make ip)
├── Makefile               # Top-level build orchestration
└── Solution/Makefile      # ARM software build
```

## Prerequisites

- Vitis HLS 2022.2
- Vivado 2022.2
- ARM cross-compiler (`arm-linux-gnueabihf-g++`) for the software build
- Pynq board with Linux image, accessible over SSH

## Build Steps

### 1. Synthesise HLS IPs

```bash
make ip
```

Runs both HLS TCL scripts via Vitis HLS. Exports:
- `IP-repo/Conv2D_HW/` — Conv2D accelerator IP
- `IP-repo/MaxPool_HW/` — MaxPool accelerator IP

To build individually:
```bash
make ip_conv     # Conv2D only
make ip_maxpool  # MaxPool only
```

### 2. Create Vivado Block Design and Generate Bitstream

```bash
make vivado_project   # unzips IPs and creates the Vivado project
make bitstream        # runs synthesis, implementation, bitstream generation
```

The block design connects:
- Conv2D HP0 (master1: input/output), HP1 (master2: filters/biases)
- MaxPool HP2 (gmem0: input), HP3 (gmem1: output)
- AXI control: Conv2D at `0x40000000`, MaxPool at `0x40010000`

### 3. Build ARM Software

```bash
cd Solution && make
```

Produces `cnnSolver` binary (cross-compiled for ARM).

### 4. Deploy and Run on Pynq

Copy to the board:
```bash
scp Solution xilinx@<board-ip>:~
scp <bitstream>.bit     xilinx@<board-ip>:~
scp <bitstream>.bit     xilinx@<board-ip>:~
```

Program the FPGA and run:
```bash
sudo fpgautil -b <bitstream>.bit
sudo ./cnnSolver <image>.rgba.planar
```
### 5. Clean

```bash
make clean          # remove HLS projects and IPs
